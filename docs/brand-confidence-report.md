# Sinu Aed — Brand Confidence Report

> Compiled: 2026-04-12
> Sources: Chain API brand frequency (3 chains × 25 garden search terms), Google organic results, Hind.ee price comparison, Estonian gardening forums, manufacturer sites.

## Methodology

Three data sources triangulated:
1. **Chain API brand frequency** — searched 25 garden terms across Decora, Ehituse ABC, Espak. Counted brand appearances + chain presence.
2. **Google organic results** — searched product keywords in Estonian. Brands that appear in top results = what consumers search for.
3. **Hind.ee** — Estonian price comparison aggregator. Products listed there = products people actually compare prices on.
